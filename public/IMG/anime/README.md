# Imágenes Anime

Esta carpeta debe contener las imágenes anime correspondientes a los arquetipos.

## Imágenes requeridas

- `hero-anime.jpg` - Imagen principal para el hero section en modo anime
- `anfitriona-anime.jpg` - Arquetipo Anfitriona (estilo anime)
- `ejecutiva-anime.jpg` - Arquetipo Ejecutiva (estilo anime)
- `musa-anime.jpg` - Arquetipo La Musa (estilo anime)
- `porrista-anime.jpg` - Arquetipo Porrista/Entusiasta (estilo anime)

## Recomendaciones

- Resolución mínima: 800x800px
- Formato: JPG o PNG
- Estilo: Personajes anime profesionales, similares a los arquetipos realistas
- Calidad: Alta resolución para buena visualización

## Fuentes sugeridas

Puedes buscar imágenes en:
- Pixabay (búsqueda: "anime girl character")
- Unsplash (búsqueda: "anime portrait")
- Pexels (búsqueda: "anime character")

**Importante**: Asegúrate de que las imágenes tengan licencia de uso comercial libre o que tengas los derechos necesarios.
